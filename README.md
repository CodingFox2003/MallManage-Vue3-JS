### 项目简介：

基于`Vue3` + `ElementPlus` + `Vite`构建的现代化商城后台管理系统。该系统提供了完整的电商后台管理功能，包括商品管理、订单处理、用户管理、权限控制等核心模块，采用前后端分离架构，通过Mock.js模拟后端数据，可作为学习Vue3及其生态系统应用的完整案例。

### 功能简介：

- **权限管理系统**：支持多角色用户权限分配，动态菜单渲染，细粒度权限控制
- **商品管理模块**：商品分类管理、商品信息维护、库存管理、商品状态控制
- **订单处理系统**：订单全流程管理（待付款、待发货、待收货、已完成、已取消）、退款申请处理
- **用户管理中心**：用户信息管理、用户等级设置、店铺管理与审核
- **数据统计展示**：首页数据概览、销售数据可视化展示
- **响应式设计**：适配不同屏幕尺寸，提供良好的用户体验

### 采用技术

- `Vue3.2 + Vue-router4 + Vuex4 + Vite2 + Vueuse`
- `ElementPlus`
- `Naive-ui`
- `Windicss`

### 使用简介

#### 安装依赖

必须使用`yarn`或`yarn install`安装依赖
```bash
yarn or yarn install 
```

#### 项目运行

```bash
yarn dev
```

#### 项目打包

```bash
yarn build
```
